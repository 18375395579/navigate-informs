Title: DICE: Software for Collaborative Engineering
Author: Suvrajeet Sen, Professor, The Ohio State University, 1971 Neil Ave, Columbus, OH, 43210, United States of America, sen.22@osu.edu
Coauthor(s): Shugang Kang, Yunwei Qi
Year: 2008
Abstract: DICE is a collaborative environment which allows a engineers to gain insights into a system by interacting with a data using natural language queries. The software provides a retrospective understanding of the system through its database, and a predictive outlook of the system using modeling and simulation.
