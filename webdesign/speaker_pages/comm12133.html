Title: A Model for Open Source Project Performance Using Data Mining and Text Mining

Author: Uzma Raja, University of Alabama, Box 870226, Tuscaloosa, AL, 35487-0226, United States, URaja@cba.ua.edu

Coauthor(s): Marietta Tretter

Year: 2006

Abstract: A model for identifying the factors that affect the performance of Open Source Software (OSS) projects was developed using Data Mining and Text Mining. The OSS project data from Source Forge repository was used to develop test and validate the model. The model was improved using new variables created by using Text Mining techniques.
