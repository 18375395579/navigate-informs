Title: System Optimum Assignment of Wireless-Equipped Assets: Problem Variations and Solution
Author: Natalia Ruiz Juri, natiruizjuri@gmail.com
Coauthor(s): S. Travis Waller
Year: 2006
Abstract: This work explores the marginal cost of routing wireless-equipped assets into a stochastic network, as a means to analyze and quantify information availability impacts on total system expected cost. Optimization models are proposed for several problem variations, involving different assumptions and objectives. Solution algorithms are developed and tested in a number of example networks. The result analysis provides useful insights about the problem significance and complexity.
