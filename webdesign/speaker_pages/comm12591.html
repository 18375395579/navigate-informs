Title: Urban Freight Performance Measurement Using GPS Data
Author: Xia Yang, Urban Freight Performance Measurement Using GPS Data, RPI, TROY, NY, 12180, United States of America, yangx5@rpi.edu
Coauthor(s): Jose Holguin-Veras, Xuegang Ban, Jeff Wojtowicz
Year: 2013
Abstract: We present methods for using GPS data to measure urban freight performances. Case studies will be conducted using data from major metropoltian areas.
