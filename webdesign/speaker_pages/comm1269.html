Title: Automated Tuning of Solver Parameters

Author: Mustafa Baz, University of Pittsburgh, 5837 Hobart Street Apt. #2, Pittsburgh, PA, United States, mub3@pitt.edu

Year: 2006

Abstract: We present automated tuning algorithms and implementations which determine good parameter settings for particular instances for MIP solvers. We compare several approaches using ideas from software testing and machine learning. Some of the ideas we use are pairwise testing, decision trees, and artificial neural networks. Our code is available under an open source license.
