Title: Improved Sensor Netting Using Radar Target Object Map

Author: George Mayernik, Raytheon Company, 235 Presidential Way, MS 26/2207c, Woburn, MA, 01801-1060, United States of America, George_E_Mayernik@raytheon.com

Year: 2008

Abstract: Radar netting techniques developed and evaluated using Radar Target Object Map (RTOM) applied with the Tactical Component Network are described. The RTOM is presented with an illustration of the approach. Applications of RTOM are described and illustrated using track data fusion of X-Band Radars. It is shown that the use of RTOM provides a key ingredient to allow sensors to work as a unit in a coordinated manner with direct communications between them and with the fire units under C2BMC control.
