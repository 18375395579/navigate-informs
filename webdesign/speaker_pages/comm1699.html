Title: Portfolio Optimization for Investment in Global Markets, using Markowitz’s Portfolio Selection Model
Author: Jongbin Jung, Yonsei University, 50 YonseiRo, Seodaemungu, Seoul, 120-749, Korea, Republic of, olorin@yonsei.ac.kr
Coauthor(s): Seongmoon Kim
Year: 2012
Abstract: This paper investigates portfolio optimization in various global markets. The proposed model is based on Markowitz’s portfolio selection model, uses an exponential smoothing forecast method to improve parameter estimates, and incorporates fine-tuned adjustments for different markets around the world. The performance of the model is evaluated against benchmark indices in each respective market. (e.g. Dow Jones Industrial Average, FTSE).
