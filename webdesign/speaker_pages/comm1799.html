Title: Using Sensor Data and Reliability Analysis to Improve Aircraft Component Failure Predictions

Author: Johnathon Dulin, Concurrent Technologies Corporation, 771 Fairdale Ct, Castle Rock, CO, 80104, United States of America, dulinj@ctc.com

Coauthor(s): Alan Moses

Year: 2012

Abstract: Sensor data use in Army Aviation is generally limited to determining when a component exceeds a maintenance threshold. With a wealth of data available from onboard sensors, those data could better support maintenance and sustainment decisions if used effectively. We use sensor data and operational profiles to project when a component will cross its thresholds, considering component and supply chain uncertainty to predict when a component will require replacement and to trigger a demand signal.
