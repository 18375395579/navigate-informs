Title: Timetabling with TS-OPT

Author: Thomas Schlechte, Research Assistant, Zuse Institute Berlin, Takustrasse 7, Berlin, 14195, Germany, schlechte@zib.de

Coauthor(s): Ralf Borndoerfer

Year: 2008

Abstract: This talk focuses on the macroscopic train timetabling problem, which asks for a conflict free train schedule of maximum value for a given railway infrastructure. We present our optimization module TS-OPT, which solves this problem using sophisticated column generation techniques. Finally, we present first preliminary results on a part of German long distance area; tttplib.zib.de provides these simplified realistic data sets.
