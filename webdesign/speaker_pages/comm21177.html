Title: An Active Set SQP Algorithm for Nonlinear Programming with Inexact Subproblem Solutions

Author: Travis Johnson, PhD Candidate, Northwestern University, Engineering Science/Applied Math, 2145 Sheridan Rd, Evanston, IL, 60201, United States of America, traviscj@traviscj.com

Coauthor(s): Daniel Robinson, Andreas Waechter, Frank Curtis, Christian Kirches

Year: 2013

Abstract: We discuss theory, implementation, and integration of two algorithms: a nonlinear continuous optimization algorithm which allows reliable use of inexact subproblem solutions while also using steering rules to handle infeasible problem instances and an iterative quadratic programming solver which reuses an existing factorization of a related KKT matrix(e.g. from previous SQP iterations) to compute approximate subproblem solutions. We report extensive numerical results.
