Title: Optimica- An Extension of Modelica Supporting Dynamic Optimization
Author: Johan Akesson, PhD, Lund University, Box 118, Lund, 221 00, Sweden, johan.akesson@control.lth.se
Year: 2008
Abstract: An extension of Modelica (a modeling language for dynamic systems), entitled Optimica is presented. Optimica supports formulation of dynamic optimization problems based on Modelica models. A prototype compiler which supports a subset of Modelica and a first draft version of Optimica is under development. The compiler also supports code generation to AMPL, including automatic transcription of continuous time derivatives. The resulting algebraic problem is solved by AMPL/IPOPT.
