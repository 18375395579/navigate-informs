Title: The Branch-and-Sandwich Algorithm for Bi-Level Problems: Implementation and Computational Results

Author: Polyxeni-M. Kleniati, Dr., Imperial College London, Department of Chemical Engineering, South Kensington Campus, London, SW7 2AZ, United Kingdom, polyxeni.kleniati03@imperial.ac.uk

Coauthor(s): Claire S Adjiman

Year: 2012

Abstract: The Branch-and-Sandwich algorithm (Kleniati and Adjiman, J. Global Optim., 2012) is a deterministic global optimization algorithm developed for optimistic bi-level programming problems with a nonconvex inner problem satisfying a regularity condition. Practical considerations, algorithmic options and performance are explored via a C++ implementation of the algorithm using the MINOTAUR open-source branch-and-bound toolkit (Leyffer, Mahajan, Munson, Linderoth and Luedtke).
