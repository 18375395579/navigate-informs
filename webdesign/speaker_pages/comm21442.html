Title: Dynamic Programming for Bayesian Reinforcement Learning

Author: Pascal Poupart, Assistant Professor, University of Waterloo, 200 University Avenue West, Waterloo, ON, N2L 3G1, Canada, ppoupart@cs.uwaterloo.ca

Coauthor(s): Nikos Vlassis

Year: 2007

Abstract: Dynamic programming can be used to optimize the policy of Bayesian reinforcement learning problems. However, since the form of the value function is unknown, existing techniques tend to avoid dynamic programming. We derive an analytic solution, revealing that the optimal value function is the upper envelope of a set of linear combinations of Dirichlets. We also propose an effective approximate dynamic programming algorithm, called Beetle, which takes advantage of this analytical form.
