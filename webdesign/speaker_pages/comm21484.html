Title: A Cutting Plane Algorithm for Solving a Robust Edge Partitioning Problem

Author: Zeki Caner Taskin, Ph.D Student, University of Florida, University of Florida, ISE Department, 303 Weil Hall, Gainesville, FL, 32611-6595, United States, taskin@ufl.edu

Coauthor(s): Shabbir Ahmed, J. Cole Smith, Andrew Schaefer

Year: 2007

Abstract: The edge partitioning problem is inspired by a SONET design problem, and considers the partitioning of a graph's edge set into a set of subgraphs. Each subgraph is limited by the (weighted) sum of edges and number of nodes assigned to it. This talk considers a stochastic version of the problem in which there are several scenarios regarding edge weights. We formulate this problem as an MIP, and prescribe a two-stage cutting-plane approach with binary variables in both stages.
