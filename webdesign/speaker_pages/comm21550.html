Title: Pyomo: Python Optimization Modeling Objects

Author: William Hart, Sandia National Laboratories, PO Box 5800, MS 1318, Albuquerque, NM, 87185, United States of America, wehart@sandia.gov

Coauthor(s): Leonardo B. Lopes

Year: 2008

Abstract: We describe the Python Optimization Modeling Objects (Pyomo) package. Pyomo is a Python package that provides an optimization modeling capability that is commonly associated with algebraic modeling languages like AMPL and GAMS. Pyomo can be used to define abstract problems, create concrete problem instances, and solve these instances with standard solvers. The Python scripting language provides a clean syntax for defining models while providing the flexibility of a general programming language.
