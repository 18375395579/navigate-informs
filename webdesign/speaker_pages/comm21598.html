Title: Restrict-and-Relax Search for 0-1 Integer Programs

Author: Menal Guzelsoy, SAS, 100 SAS Campus Drive, Cary, NC, 7513, United States of America, menal.guzelsoy@sas.com

Coauthor(s): George Nemhauser, Martin Savelsbergh

Year: 2012

Abstract: We introduce restrict-and-relax search, an algorithm for 0-1 integer programs that explores a dynamic search tree by not only fixing variables (restricting), but by also unfixing previously fixed variables (relaxing). Starting by solving a restricted integer program, we may at any tree node selectively relax/restrict variables using dual/structural information. A proof-of-concept computational study demonstrates the effectiveness of the algorithm.
