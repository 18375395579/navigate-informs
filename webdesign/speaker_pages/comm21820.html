Title: Warm Starting for Mixed Integer Linear Programs

Author: Menal Guzelsoy, Lehigh University, 200 West Packer Avenue, Bethlehem, PA, 18015, United States of America, megb@lehigh.edu

Coauthor(s): Ted Ralphs

Year: 2008

Abstract: Warm starting is a technique used to start the solution procedure of a modified mixed integer linear program (MILP) from an advanced state using data generated from execution of a previous solution procedure. In this talk, we discuss warm starting procedures that can be used with branch-and-cut algorithms for MILPs and analyze techniques for obtaining effective starting points.
