Title: SCIP - A Framework for Constraint Integer Programming
Author: Tobias Achterberg, Ph.D, ILOG, Ober-Eschbacher Strasse 109, Bad Homburg, 61352, Germany, tachterberg@ilog.de
Year: 2007
Abstract: SCIP integrates techniques from MIP, CP, and SAT to solve constraint integer programs. User plugins can provide additional functionality like new constraint types, cut separators, or branching rules. All plugins needed to solve MIPs are included in the distribution. SCIP is available in source code and free for academic and non-commercial use. We give an overview of the software and present benchmark results.
