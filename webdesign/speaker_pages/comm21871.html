Title: Setting Up and Hosting Your Solver as Web Services via Optimization Services (OS)
Author: Jun Ma, Northwestern University, Dept. of Industrial Eng & Mgmnt Sciences, 2145 Sheridan Road, Evanston, IL, 60208-3119, United States of America, maj@northwestern.edu
Coauthor(s): Robert Fourer, Kipp Martin
Year: 2007
Abstract: Optimization Services (OS) is a unified framework for new generation distributed optimization systems. We provide the open source OS library which is based on a set of Optimization Services Protocols (OSP). The library supports the solution of a wide variety of optimization problem types in a loosely coupled distributed environment. We also provide OS server software for users to host their own Optimization Services. In particular, we demo the OS integration in Apache's Tomcat Web server.
