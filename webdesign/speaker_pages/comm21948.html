Title: An Approximate Dynamic Programming Approach to Network Revenue Management with Customer Choice

Author: Dan Zhang, University of Chicago, 5807 S. Woodlawn Avenue, Chicago, IL, 60637, United States, dan.zhang@ChicagoGSB.edu

Coauthor(s): Dan Adelman

Year: 2006

Abstract: We consider a network revenue management problem where customers choose among open fare products according to some pre-specified choice model. We approximate the dynamic programming value function with an affine function of the state vector, and show how to solve the resulting linear program using column generation.
