Title: Equitable Bandwidth Allocation in Content Distribution Networks

Author: Hanan Luss, Senior Scientist, Telcordia Technologies, One Telcordia Drive, RRC-1D254, Piscataway, NJ, 08854, United States of America, hluss@telcordia.com

Year: 2008

Abstract: We present a bandwidth allocation model for a network that consists of multiple trees, where the root of each tree has a server that broadcasts multiple programs. The model is formulated as an equitable resource allocation problem with a lexicographic maximin objective function. The constraints include link capacity constraints and tree-like ordering constraints. We present an algorithm that provides an equitable solution in polynomial time for certain performance functions.
