Title: Aircraft Stand Allocation with Associated Resource Scheduling

Author: Tor Fog Justesen, Industrial PhD Student, Copenhagen Airports A/S, Lufthavnsboulevarden 6, PO Box 74, Kastrup, 2770, Denmark, tor.justesen@cph.dk

Coauthor(s): Richard Lusby, Jesper Larsen, David Ryan

Year: 2012

Abstract: When handling an aircraft turnaround different ground handling resources are required at different times. Each resource can be claimed by at most one turnaround at a time. In this project we develop a set partitioning model formulation of the aircraft stand allocation problem with associated resource scheduling and an optimization based solution algorithm in which knowledge about the structure of the problem is exploited to generate good feasible solutions.
