Title: Bilevel Mixed-integer Interdiction Model and Exact Algorithm for Influence Dominating Set Problem

Author: Soheil Hemmati, PhD Student, University of Florida, 406 Weil Hall, Gainesville, FL, 32603, United States of America, soheilmn@ufl.edu

Coauthor(s): J. Cole Smith

Year: 2012

Abstract: Social networks, computer networks, etc., are examples of real-world networks in which an attacker may try to exploit the network structure in order to propagate some type of influence such as rumors, infections, etc. over the nodes. We consider a scenario in which we can protect the nodes from being the source of such malicious influence propagation. We suggest a bilevel mixed-integer program, a set of valid inequalities, and an exact algorithm tailored for the problem.
