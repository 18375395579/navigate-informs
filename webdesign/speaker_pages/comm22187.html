Title: Branchwidth via Integer Programming

Author: Illya Hicks, Associate Professor, Texas A&M University, Industrial and Systems Engineering MS 3131, Zachry Engineering Center, College Station, TX, 77843-3131, United States, ivhicks@tamu.edu

Coauthor(s): Elif Kolotoglu

Year: 2006

Abstract: Branch decomposition based algorithms have been used in a practical setting to solve NP-hard problems modelled on graphs. The essential element of a branch decomposition based algorithm is an input branch decomposition whose width is close to the branchwidth for the input graph. We present an integer programming formulation for solving the optimal branch decomposition and branchwidth problem. Some preliminary computational results are also presented.
