Title: Efficient Multidimensional, Multiclass Classification for Mission Critical Applications

Author: Tianyi Wu, Research Assistant, University of Illinois, Urbana-Champaign, 201 N Goodwin Ave, Urbana, IL, 61801, United States, twu5@uiuc.edu

Coauthor(s): Kaizhi Tang, Atul Phadke, Jiawei Han

Year: 2007

Abstract: The classification on numerical data sets with many features is often multidimensional and multiclass. It is hard because each training set contains multi-dimensional and multi-level class labels. Previous methods use binary classifiers and cannot handle it well. We propose a new method by exploiting the correlation among class dimensions. We cluster class labels based on the correlation and use SVM to obtain the classifier. The experiment on a biodynamic database shows our method is effective.
