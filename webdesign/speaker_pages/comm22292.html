Title: Target-tracking Model for Independent Sensors

Author: Eduardo Pasiliao, AFRL Munitions Directorate, 101 West Eglin Boulevard, Eglin AFB, FL, 32542, United States, eduardo.pasiliao@eglin.af.mil

Year: 2007

Abstract: The presentation introduces an integer programming model with a maximum likelihood function for the data association problem in track fusion. The multitarget multisensor problem has traditionally been formulated as a multidimensional assignment problem, which has a complexity that grows exponentially with the number of sensors. Our model relies on solving a series of two-dimensional assignment problems to give it a complexity that only grows linearly.
