Title: Improved Techniques for Programming Contingent Logic in Spreadsheets

Author: Thomas Grossman, tagrossman@usfca.edu

Year: 2008

Abstract: Programming complicated logic in spreadsheets is troubling because programmers rely heavily on complex nested IF statements that are difficult to program and more difficult to verify. We share some real-world examples of complex nested IF statements, and present spreadsheet programming techniques to replace them them with code that is more transparent, easier to program, and easier to verify.
