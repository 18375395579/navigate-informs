Title: MIPing the Probability Set Covering Problem

Author: Vineet Goyal, PhD Student, Tepper School of Business, Carnegie Mellon University, 5000 Forbes Avenue, Pittsburgh, PA, 15213, United States, vgoyal@andrew.cmu.edu

Coauthor(s): Anureet Saxena, Miguel Lejeune

Year: 2006

Abstract: We consider a probabilistic set covering problem where the right hand side is a random binary vector and the covering constraint must be satisfied with some probability. The problem can be formulated as a disjunctive program over the set of p-efficient points, and is solved with a branch and bound algorithm. In this paper, we give a novel MIP formulation of the probability constraint using only few additional variables. This approach overcomes the exponential enumeration of p-efficient points.
