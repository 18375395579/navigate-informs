Title: Quantitative Object Reconstruction Using X-Ray Tomography and Mixed Variable Optimization
Author: Mark Abramson, Air Force Institute of Technology, 2950 Hobson Way, AFIT/ENC, Building 641, Wright-Patterson AFB, OH, 45432, United States, Mark.Abramson@afit.edu
Coauthor(s): John Dennis, Thomas Asaki, Rachael Pingel, Kevin O'Reilly
Year: 2006
Abstract: A new approach for quantitatively reconstructing objects from x-ray tomography radiograph data is introduced. A mixed variable programming (MVP) problem is formulated, in which the decision variables are the number and types of materials and the thicknesses of each layer. The objective is to minimize the difference between the data and an Abel transform model of the configuration. Numerical results, using the mixed variable pattern search algorithm are presented for several test configurations.
