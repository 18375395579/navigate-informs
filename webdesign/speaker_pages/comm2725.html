Title: A Very Large Scale Search Algorithm for General Balancing Problems

Author: Bob Storer, Professor, ISE Department, Mohler Lab 200, Lehigh University, Bethlehem, PA, 18015, United States, rhs2@lehigh.edu

Year: 2006

Abstract: We discuss a Very Large Scale Neighborhood search algorithm for solving a generalized balancing problem. At each iteration an extension of the number partitioning problem is solved. This work extends a previous, quite successful algorithm for the rotor blade balancing problem.
