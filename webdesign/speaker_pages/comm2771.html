Title: Pre-Allocating Multi-Component Available-to-Promise Quantities Using a Stochastic Programming Model

Author: Chien-Yu Chen, Assistant Professor, George Mason University, 4400 University Drive, Fairfax, VA, 22030, United States, cchen8@gmu.edu

Year: 2006

Abstract: To support real-time order promising, an available-to-promise (ATP) system allocates available assemble-to-order (ATO) components before knowing actual customer orders. Assuming discrete distributions of future demands, we pre-allocate multiple ATO components into distinct time-product buckets by using a stochastic linear programming (SLP) model. The two-stage SLP model is solved with L-shaped decomposition method. The SLP solution is tested against simulated continuous demand distributions.
