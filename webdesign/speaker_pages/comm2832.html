Title: Data Aggregation Method for A Covering Problem with Gradual Coverage
Author: Hoon Jang, KAIST, 335 Gwahangno, Yuseong gu, Daejeon, Korea, Republic of, hoon.jang@kaist.ac.kr
Coauthor(s): Taesik Lee
Year: 2013
Abstract: We study a data aggregation method for covering location problem. There exists an aggregation method that reduces data without causing errors. It was developed for covering problem with 0-1 type coverage, and we generalize the method to apply to gradual coverage covering problem. We approximate a gradual coverage function into a step-wise function using K-means algorithm, and a common reachability set is constructed accordingly. Experimental results show the effectiveness of the proposed method.
