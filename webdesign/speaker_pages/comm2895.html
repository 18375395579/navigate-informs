Title: Approximating the Value Function of a Mixed Integer Linear Program

Author: Menal Guzelsoy, Lehigh University, 200 West Packer Avenue, Bethlehem, PA, 18015, United States of America, megb@lehigh.edu

Coauthor(s): Ted Ralphs

Year: 2008

Abstract: The value function of a mixed integer linear program (MILP) is a function that returns the optimal solution value as a function of the right-hand-side. In this study, we approximate the value function of a MILP by considering various subadditive approximations to the value functions derived from single row relaxations using results from MILP duality and the technique of maximal subadditive extension.
