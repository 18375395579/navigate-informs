Title: JEGA: A Tool for Multi-Objective Optimization

Author: John Eddy, Senior Member of the Technical Staff, Sandia National Laboratories, Mail Stop 1125, PO Box 5800, Albuquerque, NM, 87185-1125, United States, jpeddy@sandia.gov

Year: 2006

Abstract: JEGA is a software package that implements a multi-objective genetic algorithm (MOGA) for solution to multi-objective problems. JEGA is highly configurable and provides a rich set of algorithmic components for tailored optimization. It has been used successfully on many problems by users both internal and external to Sandia.
