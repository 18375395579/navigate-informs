Title: Exploration vs. Exploitation in the Information Filtering Problem
Author: Xiaoting Zhao, Cornell University, Ithaca, NY, 14850, United States of America, xz337@cornell.edu
Coauthor(s): Darlin Alberto, Peter Frazier
Year: 2013
Abstract: In information filtering, a computer filters irrelevant items from a stream of information too voluminous to process by hand (e.g., scientific papers, blog posts, satellite photos). Such systems face the exploration vs. exploitation tradeoff, in which it may be beneficial to present an item that is likely to be irrelevant, because feedback allows the system to learn and improve. We present an analysis of this problem using stochastic dynamic programming, and an application to the arXiv.
