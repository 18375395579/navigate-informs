Title: Web User Behavioral Profiling for User Identification

Author: Catherine Yang, UC Davis, yiyang@ucdavis.edu

Year: 2008

Abstract: In this paper, we propose a simple, yet powerful approach to profile users Web browsing behavior for the purpose of user identification. The importance of being able to identify users can be significant given a wide variety of applications in electronic commerce, such as product recommendation, personalized advertising, etc. We create user profiles capturing the strength of users behavioral patterns, which can be used as behavior signatures to help in identifying users.
