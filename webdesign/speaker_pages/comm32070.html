Title: Improving Railroad Service Design through Analytics

Author: Todd Workman, Canadian Pacific, Calgary, AB, Canada, todd_workman@cpr.ca

Coauthor(s): Ravindra Ahuja, MohammadReza Baharnemati

Year: 2013

Abstract: Canadian Pacific Railway (CPR) generates a lot of data related to shipment, block, and trains, which is stored in our data warehouses. Accessing and analyzing this large amount of data in a timely manner has been a challenge for us. To address this, we have recently built a data analytics system. In this presentation, we will give an overview of this system and describe how we are using this system to answer various questions faced by the service design department and create better train plans.
