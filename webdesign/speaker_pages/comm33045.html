Title: Empirical Bug Analysis in Apache Virtual Computing Lab

Author: Rahul Ghosh, Advisory Software Engineer, IBM, 4205 South Miami Blvd, Durham, NC, 27703, United States of America, rghosh@us.ibm.com

Coauthor(s): Flavio Frattini, Marcello Cinque, Andy Rindos, Stefano Russo, Kishor Trivedi

Year: 2013

Abstract: Open source Cloud software stacks (e.g., OpenStack, Apache VCL) have recently gained significant attention both from industry and academia. Understanding the software bugs in such Cloud platforms is an important but rarely investigated issue till date. We performed an analysis of the reported bugs in Apache VCL. Results help in identifying the defect prone components and phases in Cloud and assist in the software testing and debugging stage.
