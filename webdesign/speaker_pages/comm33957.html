Title: Tactics for Structuring Big Data for Analysis
Author: Helen (Xianzhi) Wang, Quantitative Analyst, Google, 1600 Amphitheatre Parkway, Mountain View, CA, 94043, United States of America, xzwang@gmail.com
Year: 2013
Abstract: Cloud and big data open source technologies have made big data analytics more approachable. In order to effectively use big data to draw insights and make better decisions for the business, it is important to analyze then structure the data for the intended purpose. This talk aims to share some tactics to handle big data tables so that it becomes more accessible to analytic models.
