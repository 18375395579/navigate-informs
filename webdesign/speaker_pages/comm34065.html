Title: Performance of the Markowitz’s Portfolio Selection Model over the Accuracy of Estimated Values

Author: Hongseon Kim, Yonsei University, 50 YonseiRo, Seodaemungu, Seoul, Korea, Republic of, caasman@gmail.com

Coauthor(s): Seongmoon Kim

Year: 2012

Abstract: Using financial data in the Korean stock market, this paper empirically analyzes how the difference between estimated and actual values of each stock's expected return, standard deviation, and correlation coefficient might affect overall risk and return of the portfolio. We found that increasing accuracy of the input data, especially, each stock's expected return, had a significant impact on the return of the portfolio.
