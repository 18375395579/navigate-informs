Title: Automatic Parameter Tuning for ILOG CPLEX

Author: Mary Fenelon, ILOG, Inc., 889 Alder Avenue, Incline Village, NV, 89451, United States, mfenelon@ilog.com

Coauthor(s): Zonghao Gu

Year: 2007

Abstract: The default values of the many CPLEX parameters provide good performance over a wide variety of problems but there may be better values for a particular set of problems. CPLEX is now providing a tool which automatically tunes the parameter values for a set of problems. This presentation will cover some of the decisions made in the design of this feature and some typical results.
