Title: An Integer Programming Approach to Determine Maximum Ambiguity in Fault Tolerant Sensor Networks

Author: Sibel B. Sonuc, PhD Candidate, University of Florida, PO Box 210020, Gainesville, FL, 32611, United States of America, sibel.bilge@ufl.edu

Coauthor(s): J. Cole Smith

Year: 2012

Abstract: We study fault tolerant sensor networks where among a set of sensors deployed, a subset of them might fail. A faulty sensor may or may not report a false reading. As a result, the network owner may not be able to clarify the status of every node. A node is called an ambiguous node, if it is not possible to clarify the status of that node with the current sensor reports. We propose an integer programming approach for finding the maximum number of ambiguous nodes subject to the sensor reports.
