Title: Analyzing the Sequential Stochastic Multilevel Passenger Screening Problem

Author: Adrian Lee, University of Illinois, Urbana, IL, United States, ajlee4@uiuc.edu

Coauthor(s): Laura McLay, Sheldon H. Jacobson

Year: 2007

Abstract: The problem of assigning sequentially arriving passengers to a multilevel aviation security screening system in real-time is formulated as a discrete-time set of difference equations. Using feedback linearization, an optimal state feedback control algorithm is developed to achieve desired security class occupancies while maximizing a total system security measure.
