Title: In-network Function Computation: Spanning Trees and Multi-class Fork-join Queues

Author: Samyukta Sethuraman, Texas A&M University, 3131 TAMU, College Station, TX, United States of America, samyukta@tamu.edu

Coauthor(s): Natarajan Gautam

Year: 2013

Abstract: We consider a sensor network where nodes simultaneously measure events and transmit to a sink. However, the sink node is ultimately interested in a function of the measured values, such as maximum, minimum, etc. To efficiently perform this under stochastic conditions, we seek a set of spanning trees to route the sensed information. We model the flow of information as a fork-join queueing network. We obtain bounds on the performance, and thereby select ideal set of spanning trees.
