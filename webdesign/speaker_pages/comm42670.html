Title: Demand Forecasting and Resource Planning Utilizing the Operating Plan Tool Set at CSX

Author: Gabe Farra, CSX Transportation, 500 Water St. J250, Jacksonville, FL, 32202, United States, Gabe_Farra@csx.com

Coauthor(s): Dharma Acharya

Year: 2007

Abstract: CSX is utilizing an in-house forecasting tool to input an OD level demand forecast. The Operating Plan Tool (OPT) is then fed the forecasting output to develop resource requirements for crew hiring, car demand, locomotive demand and capacity enhancements. The presentation will describe the process by which OD level demand is input and fed into the OPT tool to develop the operating train plan that allows for developing resource requirements.
