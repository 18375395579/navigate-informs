Title: Fast Computation of Point-to-point Time-dependent Paths on Dynamic Road Networks

Author: Giacomo Nannicini, LIX, Ecole Polytechnique, Palaiseau, NA, France, giacomo.nannicini@v-trafic.com

Coauthor(s): Philippe Baptiste, Leo Liberti, Daniel Krob

Year: 2007

Abstract: We propose a Polynomial-Time Approximation Scheme for the point-to-point time-dependent shortest path problem, using which good solutions may be quickly found in large graphs. This is useful for a typical server scenario, where the computation of point-to-point shortest paths in large road networks with time-dependent arc costs (travelling times) is needed, and each path has to be computed in a very short time. Our approach includes a time-consuming pre-processing phase and a fast query phase.
