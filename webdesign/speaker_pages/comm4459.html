Title: The Page Ranking and Path Analysis of Web Site Traffic

Author: Mark Lewis, Associate Professor, Missouri Western State University, mlewis14@missouriwestern.edu

Coauthor(s): Barbara Jo White

Year: 2007

Abstract: A new approach for rank ordering web pages by calculating the Most Common Paths (MCP) as created over time by web site traffic is presented. MCP refers to a sequencing of web pages into a maximal linear ordering according to the number of “votes?made by users moving to a new page from a referring one, as recorded in a transition matrix. The approach to discover the MCP utilizes server-side, anonymous, non-intrusive means to collect and analyze web user traffic.
