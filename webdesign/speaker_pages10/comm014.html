Title: Using Data Effectively in Risk Analysis using Monte Carlo Simulation

Author: Samik Raychaudhuri, Oracle Americas, 2033 N Fork Drive, Lafayette, CO, 80026, United States of America, samikr@gmail.com

Year: 2012

Abstract: In this presentation, we will discuss various roles data can play when running a Monte Carlo simulation analysis, starting from data collection, distribution modeling, and sensitivity analysis. We will also discuss a few data-cleansing procedures, and some best-practices. We will use Oracle Crystal Ball as an example implementation.
