Title: Symbol Clustering Techniques for Context-tree Models of Distributed Systems for Fault Isolation

Author: Pierre Brice, Stevens Institute of Technology, 207 Normandy Road, Edison, NJ, 08820, United States, mgi548@motorola.com

Coauthor(s): Wei Jiang

Year: 2007

Abstract: Distributed systems tend to exhibit context-dependent behavior. We model such systems using context trees. For practical applications with a large set of variables that need to be monitored for fault isolation, the set of symbols that are needed to represent the tree is extremely large. This presentation describes an information-theoretic based method to cluster these symbols and reduce the set to a manageable size.
