Title: Value Function Scales Problems in MCDM Portfolio Analysis

Author: Adiel Almeida, Prof, Universidade Federal de Pernambuco, Cx. Postal 7462, Recife, PE, 50.630-970, Brazil, almeidaatd@gmail.com

Coauthor(s): Jônatas Almeida, Rudolf Vetschera

Year: 2013

Abstract: This work present an analysis on value function scales problems for selection of project portfolio based on multicriteria decision making (MCDM) methods. Two aggregating methods are considered: additive method and an outranking method, which uses a non-compensatory rationality, while the former corresponds to a compensatory one. Problems related to scales used for intra-criteria evaluation are considered, which is linked to inter-criteria elicitation in the case of additive methods.
