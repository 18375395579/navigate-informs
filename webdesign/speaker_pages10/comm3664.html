Title: Recent Developments in OSiL/SE

Author: Horand Gassmann, Professor, Dalhousie University, Faculty of Management, 6100 University Avenue, Halifax, NS, B3H 3J5, Canada, Horand.Gassmann@Dal.Ca

Coauthor(s): Robert Fourer, Kipp Martin, Jun Ma, Huan-Yuan Sheng

Year: 2008

Abstract: OSiL is an XML schema for the formulation of a large class of mathematical programming problems. This talk concerns recent advances in the description of stochastic programs. Also described are a parser, internal data objects and a rudimentary implementation of a decomposition algorithm.
