Title: Analysis of Join-the-Shortest-Queue Routing in Web Server Farms

Author: Mor Harchol-Balter, Associate Professor, Carnegie Mellon University, Computer Science Dept., 5000 Forbes Ave., Pittsburgh, PA, 15213, United States, harchol@cs.cmu.edu

Coauthor(s): Varun Gupta, Ward Whitt, Karl Sigman

Year: 2007

Abstract: We present the first analysis of the Join-the-Shortest-Queue task assignment policy for Web server farms. Web server farms involve Processor-Sharing servers. This work introduces a new technique: Single-Queue-Approximation (SQA), and uses the technique to prove some interesting insensitivity properties for Web server farms.
