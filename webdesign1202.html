" 2 - Adaptive Control Variates Techniques for Steady-State Simulation" Sujin Kim, Cornell University, 206 Rhodes Hall, Ithaca, NY, 14853, United States, sk329@cornell.edu" Shane Henderson" 2006"We introduce an adaptive control variate scheme for estimating steady-state performance measures for stochastic systems using simulation. We use a procedure based on sample average approximation for tuning the control variate estimators. The underlying processes are assumed to be regenerative which allows establishing consistency of these estimators in an efficient way under mild conditions."
