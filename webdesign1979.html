" 3 - On-line Steady-state Detection Based on Multiple Change-point Models Using Particle Filters" Jianguo Wu, University of Wisconsin, Milwaukee, WI, United States of America, wu45@wisc.edu" Shiyu Zhou" Xiaochun Li" Yong Chen" 2013"Steady-state detection is critical in process performance assessment, fault detection and process control. We proposed a robust on-line steady-state detection algorithm using multiple change-point model and particle filtering techniques. The steady-state detection problem is formulated as a multiple change-point problem using a segmental linear model."
