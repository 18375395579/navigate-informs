" 2 - Real-world Optimization Models Formulated and Deployed as Web-based Applications using MPL OptiMax" Sandip Pindoria, Maximal Software, Ltd., Boundary House, Boston Manor, , London, W7 2QE, United Kingdom, sandip@maximalsoftware.co.uk" Bjarni Kristjansson" 2013"MPL is a modeling system that allows the model developer to efficiently formulate complicated optimization models. We will demonstrate some real-world MPL models that have been formulated as web-based applications using the MPL OptiMax Component Library. A Portfolio Optimization model will be explored and shown how it can be iterated to produce a trade-off curve (Efficient Frontier) between the Return and Risk. A Vehicle Routing application with multiple vehicles and depots. We will also demonstrate a Nurse Rostering application with multiple contract rules and objective criterias."
