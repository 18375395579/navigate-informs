Siirola" Ethan Chan" 2012"In this talk, we will describe how hybridization can be used incorporate sensitivity analysis into optimization procedures. Specifically, by dynamically incorporating Bayesian sensitivity techniques into the minimization process, we will show how the usefulness of computational models can be drastically improved. We will review the basic hybridization algorithm and resulting software. We will also give some ideas for useful hybrids and show some results for a real world problem."
